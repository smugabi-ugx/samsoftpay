Simple integration

Live in three steps

Designed for vibe coders and experienced engineers alike. Copy, paste, go.

1

Sign up and get your keys

Create an account in under 60 seconds. You immediately get test and live API keys. No approval needed to start building — test in sandbox for free.

2

Integrate with one endpoint

POST to /v1/charges with your Bearer token, an Idempotency-Key, and the customer's phone. We handle the rest — prompt delivery, status polling, webhooks.

3

Go live and get paid

Swap sk_test_ for sk_live_, complete KYC verification, and you're live. Funds settle to your bank on a rolling basis.

Simple API

Charge a customer in 10 lines

Works with Python, JavaScript, PHP, or any language that speaks HTTP.

Python JavaScript cURL
import requests, time, uuid response = requests.post( "https://yourdomain.com/v1/charges", headers={ "Authorization": "Bearer sk_live_YOUR_KEY", "Idempotency-Key": str(uuid.uuid4()), "X-Timestamp": str(int(time.time())), }, json={ "amount": 10000, # UGX 10,000 "currency": "UGX", "channel": "mtn_momo", "customer": {"phone": "256700123456"}, "reference": "order-001", } ) # → {"id": "txn_abc123", "mode": "live", "status": "authorized", "fee": 200}
