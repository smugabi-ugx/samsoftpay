Quick Start - Create a Charge

Use your API keys to initiate a collection transaction:

curl -X POST http://localhost:5000/v1/charges \
  -H "Authorization: Bearer sk_test_demo123" \
  -H "Idempotency-Key: $(uuidgen)" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 5000,
    "currency": "UGX",
    "channel": "mtn_momo",
    "customer": {"phone": "+256700123456"},
    "reference": "order-001"
  }'
